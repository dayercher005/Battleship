# Battleship

Live Preview: https://dayercher005.github.io/Battleship/

## How to Run

1. Clone the repository:
```bash
git clone 
```

2. Install dependencies
```bash
npm install
```

3. Start Development Server:
```bash
npm run dev
```

4. Open `http://localHost:8080` in your browser
